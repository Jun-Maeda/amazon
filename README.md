amazon価格スクレイピングアプリケーション
 
# DEMO
 

 
# Features
 
products.csvに登録したＵＲＬの店舗名と金額を取得してそれぞれのデータベースに保存。
最低価格を取得。
最低価格が変わっていれば通知して保存。
 
# Requirement
 
 
* requests
* bs4
* sqlite3
* datetime
* os
